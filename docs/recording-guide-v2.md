# 🎤 KK 보컬 녹음 가이드 v2

> 목표: RVC 모델 품질 대폭 개선을 위한 추가 보컬 샘플 확보

## 현재 상태
- 기존 샘플: 3개 (총 ~4분)
- 목표: **10분+ 분량** (다양한 음높이/감정)
- 모델: kk_vocal (300 에포크) → 재학습 예정 (500+ 에포크)

## 녹음 가이드

### 환경
- 조용한 방, 에어컨/선풍기 끄기
- 마이크와 입 거리: 15~20cm
- 포맷: WAV, 48kHz, 모노

### 녹음 내용 (5~7개 추천)

| # | 내용 | 목적 | 권장 길이 |
|---|------|------|-----------|
| 1 | 좋아하는 노래 1절 (한국어) | 멜로디 + 한국어 발음 | 1~2분 |
| 2 | 좋아하는 노래 1절 (영어) | 영어 발음 패턴 | 1~2분 |
| 3 | 낮은 음역 노래/허밍 | 저음 학습 | 1분 |
| 4 | 높은 음역 노래/허밍 | 고음 학습 | 1분 |
| 5 | 자유롭게 말하기 (대화체) | 말투/톤 학습 | 1~2분 |
| 6 | 감정 있는 노래 (발라드 등) | 감정 표현 학습 | 1~2분 |
| 7 | 빠른 템포 노래 (랩/팝) | 빠른 발음 학습 | 1분 |

### 주의사항
- 배경 잡음 최소화
- 클리핑(소리 깨짐) 주의 — 음량 70~80% 수준 유지
- 각 파일 사이에 2초 정도 무음 유지
- 물 마시며 목 컨디션 유지

## 파일 전달
텔레그램으로 WAV 파일 보내주시면 됩니다.
파일명 예시: `kk_vocal_v2_01_kpop.wav`

## 학습 계획
1. 새 샘플 수신
2. 48kHz 모노로 통일 전처리
3. 기존 3개 + 새 샘플 합쳐서 재학습
4. 에포크: 500 (기존 300 → 증가)
5. 피치 추출: rmvpe (기존 crepe 대비 개선)
6. 학습 후 디지털유목민 곡에 재적용 테스트
